# Analisa panel backend (Boyy99)

Sumber: panel backend Boyy99 meniru struktur menu dan dashboard dari referensi /mimin.

## Struktur utama

- **Sidebar kiri** berisi header credit dan daftar menu bertingkat.
- **Topbar** menampilkan tombol aksi cepat (DEPO 1, WD, Chat) dan ringkas user (Mnk, credit depo).
- **Konten utama** berupa grid kartu statistik harian (login, register, depo, wd, dan ringkasan nominal).
